This folder includes the code for TMLR paper: Smoothed Differential Privacy (Authors: Ao Liu, Yu-Xiang Wang, and Lirong Xia)

Please cite the above paper if you want to reuse our code.

All codes are implemented by MATLAB

The folders of exp_SGD and exp_SGD_app correspond to the experiments for SGD with Quantized Gradients in our main text and appendix respectively.

The folder of exp_voting corresponds to the experiments for Elections (in Figure 2 left and appendix)

The folder of motivating corresponds to the motivating example (Figure 1 and Appendix A)
